# Changelog

All notable changes to this project are documented in this file.

## 0.1.0-rc.1

Release candidate for first public Ribbon filter crate baseline.

Highlights:

- Standard and Homogeneous build/query modes
- Width support up to `w = 128`
- Retry + growth construction policy with structured failures
- Bit-packed filter storage and reusable query scratch
- Statistical/property/regression/fuzz quality infrastructure
